# 主题系统设计文档

## 概述

为 itools-flutter 应用实现三套独立主题系统，支持在"精致现代风"、"暗黑科技风"、"极简轻奢风"之间切换，每套主题都有亮色/暗色两种色调。

## 设计目标

- 提供三套视觉风格迥异的主题
- 每套主题支持亮色/暗色色调
- 主题切换即时生效
- 主题设置持久化存储
- 与现有备份系统集成

## 主题风格定义

### 1. 精致现代风 (Modern)
- **色彩**: 蓝紫渐变色系，柔和的过渡
- **圆角**: 12px，适中
- **字体**: 思源黑体，清晰易读
- **特效**: 毛玻璃效果、精致阴影层次
- **氛围**: 专业、精致、现代

### 2. 暗黑科技风 (Cyber)
- **色彩**: 深色背景，青绿霓虹色点缀
- **圆角**: 8px，较小
- **字体**: 等宽字体，科技感
- **特效**: 发光边框、扫描线效果
- **氛围**: 赛博朋克、未来感

### 3. 极简轻奢风 (Luxury)
- **色彩**: 米色/灰色中性色调
- **圆角**: 16px，较大
- **字体**: 衬线字体，优雅
- **特效**: 极简留白，无多余装饰
- **氛围**: 高端、优雅、克制

## 数据模型

### 主题枚举
```dart
enum AppThemeStyle {
  modern,    // 精致现代风
  cyber,     // 暗黑科技风
  luxury;    // 极简轻奢风
}
```

### 色调枚举 (已有)
```dart
enum AppThemeMode {
  light,   // 亮色
  dark,    // 暗色
  system;  // 自动（跟随系统）
}
```

### 存储格式
```json
{
  "themeStyle": "modern",
  "themeMode": "system"
}
```

存储路径: `settings/theme.json`

## 文件结构

```
lib/
  features/
    settings/
      domain/
        app_theme_style.dart      # 主题风格枚举
        theme_mode.dart           # 色调枚举 (已有)
      data/
        theme_service.dart        # 主题服务 (扩展)
      presentation/
        theme_settings_page.dart  # 主题设置页面 (重构)
  core/
    themes/
      modern_theme.dart           # 精致现代风主题配置
      cyber_theme.dart            # 暗黑科技风主题配置
      luxury_theme.dart           # 极简轻奢风主题配置
  app.dart                        # 应用入口 (修改)
```

## 主题配置结构

每套主题配置包含:
- `lightScheme` - 亮色颜色方案
- `darkScheme` - 暗色颜色方案
- `textTheme` - 文字样式
- `radius` - 圆角大小
- `buttonTheme` - 按钮样式
- `cardTheme` - 卡片样式
- `inputTheme` - 输入框样式
- 其他组件主题...

## UI 设计

### 主题设置页面
- 显示三套主题的预览卡片
- 每个卡片展示主题名称、描述、色彩示例
- 选中状态明显标识
- 底部显示色调选择（亮色/暗色/自动）

### 主题切换交互
- 点击主题卡片立即切换
- 切换时平滑过渡动画
- 保存到本地存储

## 与现有系统集成

### 备份还原
- 主题设置包含在备份中
- 还原时自动应用主题

### 多窗口
- 主窗口设置的主题同步到子窗口
- 子窗口监听主题变化

# JSON 格式化功能

## 功能概述

JSON 格式化工具提供了多种 JSON 数据处理功能，帮助开发者快速处理和转换 JSON 数据。采用左右分栏布局，左侧输入、右侧输出，支持可拖拽分割面板调整比例。

## 功能特性

### 1. 格式化

将压缩的JSON字符串格式化为易读的格式，支持自定义缩进。

**示例：**

```json
// 输入
{"name":"John","age":30,"city":"New York"}

// 输出
{
  "name": "John",
  "age": 30,
  "city": "New York"
}
```

### 2. 压缩

将格式化的JSON字符串压缩为单行，减少数据传输大小。

**示例：**

```json
// 输入
{
  "name": "John",
  "age": 30
}

// 输出
{"name":"John","age":30}
```

### 3. 转义

将JSON字符串进行转义，用于在代码中嵌入JSON。

**示例：**

```json
// 输入
{"key": "value"}

// 输出
"{\"key\": \"value\"}"
```

### 4. 反转义

将转义的JSON字符串还原。

**示例：**

```json
// 输入
"{\"key\": \"value\"}"

// 输出
{"key": "value"}
```

### 5. 智能修复

自动修复不规范或不完整的 JSON 字符串，支持以下修复策略：

- 单引号替换为双引号
- 自动补全未加引号的键名
- 移除尾部多余逗号
- 自动补全缺失的括号
- 截断至最后一个有效位置

**示例：**

```json
// 输入（不规范）
{name: 'John', age: 30,}

// 输出（修复后）
{
  "name": "John",
  "age": 30
}
```

## 界面特性

### 可拖拽分割面板

- 左右面板之间有可拖拽的分割线
- 支持鼠标拖拽调整左右面板比例（20% ~ 80%）
- 拖拽时分割线高亮显示

### 代码编辑器

- 输出面板使用 `re_editor` 代码编辑器
- 支持 JSON 语法高亮（基于 `re_highlight`）
- 显示行号
- 支持滚动条

### 实时 JSON 验证

- 输入变化时自动验证 JSON 格式（300ms 防抖）
- 状态栏实时显示验证结果（有效/无效）
- 无效时显示错误提示

## 快捷操作

| 操作     | 说明                     |
| -------- | ------------------------ |
| 格式化   | 将 JSON 格式化为易读格式 |
| 压缩     | 将 JSON 压缩为单行       |
| 转义     | 转义 JSON 字符串特殊字符 |
| 反转义   | 还原转义的 JSON 字符串   |
| 智能修复 | 自动修复不规范的 JSON    |
| 交换     | 交换输入和输出内容       |
| 复制     | 复制输出内容到剪贴板     |
| 清空     | 清空所有内容             |

## 使用场景

1. **API调试**：格式化API响应数据
2. **数据处理**：压缩JSON以减少传输大小
3. **数据修复**：智能修复不规范的JSON格式
4. **数据验证**：实时检查JSON格式是否正确

## 技术实现

- 使用 Dart 内置的 `dart:convert` 库进行 JSON 解析与处理
- 使用 `re_editor` + `re_highlight` 提供代码编辑与语法高亮
- 使用 `shadcn_ui` 组件库保持 UI 风格统一
- 支持 UTF-8 编码
- 300ms 防抖实时验证 JSON 格式
- 智能修复支持多种策略组合尝试

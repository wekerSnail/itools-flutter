# 文档导航

为降低重复维护成本，项目文档统一收敛到 `docs/` 目录。

## 目录

- `docs/tray/README.md`：系统托盘实现说明、诊断与测试清单
- `docs/deployment/README.md`：Windows 发布与部署指南（含快速排障）
- `docs/development/mirror-setup.md`：国内镜像源配置说明

## 维护约定

- 同一主题只保留一份“权威文档”（Single Source of Truth）。
- 快速参考内容优先合并为同文档的“速查”章节，而不是新增平行文档。
- 新增文档前，先评估是否应作为现有文档新增章节。
